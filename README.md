# Curvas altura e peso infantil
Projeto pessoal que utiliza dados da OMS para acompanhamento de curvas de peso e altura do meu filho.

Dados de altura por idade - WHO percentiles boys Birth to 5 years:
https://www.who.int/childgrowth/standards/cht_lhfa_boys_p_0_5.pdf?ua=1

Dados de peso por idade - WHO percentiles boys Birth to 5 years:
https://www.who.int/childgrowth/standards/cht_wfa_boys_p_0_5.pdf?ua=1
